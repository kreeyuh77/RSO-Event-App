# Online Contact Manager
This program is an event viewer wherein colleges can register their school thus allowing students to join and view events. 

**Tools**: HTML/CSS, JSON, MySQL, PHP, PHP myAdmin, LAMP Stack, Visual Studio, GitHub, Windows OS/Linux, Apache

**How To Run:**</br>
Please download 'College Event Website.sql' into your database for full experience.
